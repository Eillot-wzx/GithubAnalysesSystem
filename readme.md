## 运维部署

环境：CentOS 7.7 64位

```shell
# 安装nginx
sudo yum -y update
sudo yum install -y epel-release
sudo yum install -y nginx
# 设置nginx
sudo systemctl enable nginx
sudo systemctl start nginx

# 安装git
sudo yum -y install git

/usr/share/nginx/html

```
